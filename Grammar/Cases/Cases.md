Every noun must be comprised of a noun stem + a case at the end

There are 8 cases: The [[Absolutive Case]], [[Ergative Case]], [[Instrumental Case]], [[Allative Case]], [[Ablative Case]], [[Locative Case]], [[Vialis Case]], and [[Equalis Case]].

|       | Sing.  | Plrl.  |
| ----- | ------ | ------ |
| Abs.  | N/A    | t, -it |
| Erg.  | p, -up | t, -it |
| Inst. | mək    | nək    |
| All.  | mut    | nut    |
| Abl.  | mət    | nət    |
| Loc.  | mi     | mi     |
| Vial. | kkut   | təgut  |
| Equ.  | tut    | tut    |

The cases that begin with a nasal consonant are likely lexicalized forms of the ergative case, plus an enclitic that nasalizes the final consonant, because m is articulated in the same location as p, and likewise n with t.
